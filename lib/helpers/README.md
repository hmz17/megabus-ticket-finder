# Date Options

## Days
This is an array of numbers that stand for each day.

|Value|Weekday|
|---|-----------|
| 0 | Sunday    |
| 1 | Monday    |
| 2 | Tuesday   |
| 3 | Wednesday |
| 4 | Thursday  |
| 5 | Friday    |
| 6 | Saturday  |

## Weekends
This will auto make the `days` option `[5,6,0]`
